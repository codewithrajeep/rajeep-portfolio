# rajeep-portfolio

My personal developer portfolio built with modern web technologies.

🌐 Live: [rajeepgadal.com.np](https://rajeepgadal.com.np) `or` [https://codewithrajeep.vercel.app/](https://codewithrajeep.vercel.app/)

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Features

- Dark/light mode toggle
- Smooth page transitions
- Responsive design
- Sections: Hero, About, Experience, Projects, Skills, Contact
- Dedicated pages for full experience and projects with tech filter

## Run Locally

git clone https://github.com/codewithrajeep/rajeep-portfolio
cd rajeep-portfolio
pnpm install
pnpm dev
